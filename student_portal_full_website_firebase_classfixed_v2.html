About

Nehru Arts & Science College — Student Portal Pro

This portal is designed for college project presentation and academic record handling. It includes profile login, student details management, attendance tracking, semester-wise result analysis, grade chart visualization, and table-based PDF export for academic reports.

The system is built to make student information entry, result checking, and report generation simple, organised, and mobile friendly.

Help

  • Create your profile once, then login using the same email and password.
  • Use the Student Details module to save student information and maintain records.
  • Use the Attendance module to enter working days and present days to calculate percentage automatically.
  • Use the Student Result module to add subjects, enter marks, search class-wise results, and download the result table as PDF.
  • For viewing the same saved data on another device, login with the same account.
